This plugin will checks and if any scheduled posts have been missed, it publishes them.

== Description ==
When a visitor loads your site or you access your posts/pages admin page, this plugin checks and publishes them immediately if any scheduled posts have been missed.

== Installation ==
1. Go to Plugins, Add New, Upload Plugin.
2. Upload the ZIP file.
3. Activate the plugin through the 'Plugins' screen in WordPress

NOTE: Make sure that your timezone is set correctly in Settings->General.

== Frequently Asked Questions ==

= How often does it check missed posts? =
